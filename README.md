## ▶️ Cómo ejecutar cualquier ejercicio

**Requisitos:** tener Java instalado.
```bash
# Entrar a la carpeta del ejercicio
cd 01-adivina-numero

# Compilar
javac Principal.java

# Ejecutar
java Principal
```

## 📚 Conceptos cubiertos
- Estructuras condicionales: `if`, `else if`, `switch`
- Bucles: `for`, `while`
- Operadores: módulo `%`, división entera, potencia
- Entrada de datos con `Scanner`
- Generación de números aleatorios con `Math.random()`
- Acumuladores y contadores
